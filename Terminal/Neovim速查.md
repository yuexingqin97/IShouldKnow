## 模式切换

| 操作 | 按键 | 说明 |
|------|------|------|
| 进入 Normal 模式 | `Esc` | 任何模式下按这个回家 |
| 插入（光标前） | `i` | insert |
| 插入（光标后） | `a` | append |
| 插入（下一行） | `o` | open line below |
| 插入（上一行） | `O` | open line above |
| 追加到行尾 | `A` | Append to end of line |
| **替换模式**（逐字替换） | `R` | 每个输入字符覆盖一个原有字符，直到 `Esc` |

> **永远知道自己在哪个模式。** Normal 是默认状态，Insert 才能输入，Replace 是逐字覆盖。不确定时按 `Esc`。

### 按词 / 按行 / 特殊

| 操作 | 按键 | 说明 |
|------|------|------|
| 下一个词首 | `w` | word |
| 上一个词首 | `b` | back |
| 词尾 | `e` | end |


## 编辑

### 操作符 + 移动模型（核心心智）

```
操作符  [数字]  移动

  操作符 — 做什么（d=删, c=改, y=复制）
  数字   — 可选，重复次数
  移动   — 作用范围（w=词, $=行尾, iw=内部词）
```

### 删除

| 操作 | 按键 | 说明 |
|------|------|------|
| 删除光标处字符 | `x` | |
| 删除一个词 | `dw` | delete word |
| 删到词尾 | `de` | delete to end of word |
| 删到行尾 | `d$` 或 `D` | |
| 删除整行 | `dd` | |
| 删内部词（光标不限位置） | `diw` | delete inner word |
| 删两个词 | `d2w` | 数字放操作符和移动之间 |
| 删两行 | `2dd` | |
| 不污染寄存器 | `"_dd` | 扔黑洞 |

### 更改（删 + 自动进入 Insert）

| 操作 | 按键 | 说明 |
|------|------|------|
| 改一个词 | `cw` | change word |
| 改到词尾 | `ce` | change to end of word |
| 改到行尾 | `c$` 或 `C` | |
| 改内部词（光标不限位置） | `ciw` | change inner word —— **非常常用** |
| 替换光标字符 | `r + 新字符` | 不进 Insert 模式 |

### 撤销 / 重做

| 操作 | 按键 | 说明 |
|------|------|------|
| 撤销 | `u` | undo |
| 撤销整行全部改动 | `U` | 恢复当前行 |
| 重做 | `Ctrl + r` | redo |

## 寄存器系统（vim-01 第 3 章 + vim-02 第 2.1 章）

> vim 的剪贴板叫「寄存器」——不止一个，是可以命名的存储槽。

### 基础复制粘贴

| 操作 | 按键 | 说明 |
|------|------|------|
| 复制整行 | `yy` | yank |
| 复制一个词 | `yw` |
| 复制内部词 | `yiw` | 光标在词内任意位置即可 |
| 粘贴（光标后） | `p` | put |
| 粘贴（光标前） | `P` | |
| 复制到系统剪贴板 | `"+yy` | `+` 寄存器 = 系统剪贴板 |
| 从系统剪贴板粘贴 | `"+p` | |

> 开了 `clipboard=unnamedplus`（kickstart 默认），`yy`/`p` 直接走系统剪贴板，和 VSCode 一致。

### 命名寄存器 a-z（vim-02 2.1.1）

**刻意存到指定槽，不怕被覆盖。**

| 操作 | 按键 | 说明 |
|------|------|------|
| 复制到寄存器 a | `"ayiw` | yank inner word → 存到槽 a |
| 复制到寄存器 b | `"byy` | 存整行到槽 b |
| 删除到寄存器 s | `"sdiw` | 删内部词 → 存到槽 s |
| 从寄存器 a 粘贴 | `"ap` | |
| 在 Insert 模式下贴寄存器 a | `Ctrl + r` 然后 `a` | 不用切 Normal |

> 记忆：`"` 表示「选寄存器」→ 字母表示「哪个槽」→ 操作命令。例如 `"ayiw` = 用槽 a 复制内部词。

### 数字寄存器 0-9（vim-02 2.1.3）

**vim 自动维护的 10 条历史：**

| 寄存器 | 存什么 |
|--------|--------|
| `"0` | 最后一次 **复制**（yank）——不轻易被覆盖 |
| `"1` ~ `"9` | 最近 9 次 **整行删除**（`"1` 最新，`"9` 最老） |

- `:reg` —— 查看所有寄存器内容
- `"0p` —— 把最初 yank 的内容贴回来（中间删了一堆也不影响）

### 表达式寄存器（vim-02 2.1.2）

**在 Insert 模式下即时计算：**

| 操作 | 按键 | 说明 |
|------|------|------|
| 插入计算结果 | `Ctrl + r` `=` `60*60*24` `Enter` | 光标处插入 `86400` |
| 插入系统命令输出 | `Ctrl + r` `=` `system('date')` `Enter` | 插入当前日期 |

### 黑洞寄存器 `_`（不污染）

| 操作 | 按键 | 说明 |
|------|------|------|
| 删除不进寄存器 | `"_dd` | 扔黑洞 = `/dev/null` |

## 标记——用名字标记位置（vim-02 2.1.4）

**设置标记后，跨文件也能跳回来。不用记行号。**

| 操作 | 按键 | 说明 |
|------|------|------|
| 设标记 a | `ma` | 在当前光标位置设标记 a |
| 跳到标记 a 所在行 | `'a` | 单引号 + 标记名 |
| 跳到标记 a 的**精确位置** | `` `a `` | 反引号，比 `'a` 更精确 |

> 标记和寄存器不共享命名空间——寄存器 `a` 和标记 `a` 完全独立。

## kickstart 快捷键

> `leader` = `Space`。**不是同时按，是先后按**——先按 Space 松开（底部弹 which-key），再按后续按键。

| 功能 | 按键 | 说明 |
|------|------|------|
| **文件树** | `\` | 开关 Neo-tree |
| 模糊搜文件 | `Space` `s` `f` | Search Files |
| 全局搜内容 | `Space` `s` `g` | Search Grep |
| 搜当前词 | `Space` `s` `w` | Search Word |
| 切换已开文件 | `Space` `Space` | 双击 Space → telescope buffers |
| 格式化 | `Space` `f` | Format buffer |
| 跳定义 | `g` `r` `d` | goto definition（LSP） |
| 找引用 | `g` `r` `r` | goto references（LSP） |
| 文件内符号 | `g` `O` | document symbols |
| 诊断信息 | `Space` `s` `d` | Search Diagnostics |
