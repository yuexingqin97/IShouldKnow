# Windows → Mac 快捷键迁移

> 不是背清单，是**肌肉记忆的转换手册**。遇到"这个操作 Mac 上怎么搞"时回来查。

---

## 0. 核心差异：两个改不过来的键

在碰具体快捷键之前，先理解 Mac 键盘和 Windows 键盘的两个根本区别：

| Windows | Mac | 怎么记 |
|---------|-----|--------|
| `Ctrl` | `Cmd (⌘)` | **绝大部分快捷键把 Ctrl 换成 Cmd 就行**。Cmd 在大拇指位置，比 Windows 的小指按 Ctrl 舒服得多 |
| `Alt` | `Option (⌥)` | 名字是 Option，不叫 Alt。功能上类似但不等价——Mac 的 Option 更多用于"输入特殊字符"和"变体操作" |

> 肌肉记忆转换的第一关：把左手大拇指放到 Cmd 上，而不是小指去找左下角的 Ctrl。

## 1. 文件管理

| 操作 | Windows | Mac | 说明 |
|------|---------|-----|------|
| 打开资源管理器 / Finder | `Win + E` | 点 Dock 上的 Finder 图标，或 `Cmd + Tab` 切到 Finder | Mac 没有"一键打开 Finder"的全局快捷键，但 Finder 永远开着 |
| 重命名文件 | `F2` | `Enter`（回车） | 反直觉——回车不是打开文件，是重命名。打开文件用 `Cmd + O` 或 `Cmd + ↓` |
| 打开文件 / 进入文件夹 | `Enter` | `Cmd + O` 或 `Cmd + ↓` | |
| 返回上一级文件夹 | `Backspace` | `Cmd + ↑` | |
| 删除文件 | `Delete` | `Cmd + Delete` | 移到废纸篓 |
| 彻底删除（不进回收站） | `Shift + Delete` | `Cmd + Option + Delete` | |
| 复制文件 | `Ctrl + C` → `Ctrl + V` | `Cmd + C` → `Cmd + V` | 一样 |
| 剪切文件 | `Ctrl + X` → `Ctrl + V` | `Cmd + C` → `Cmd + Option + V` | Mac 的"剪切"是两步：先复制，粘贴时加 Option 变成移动 |
| 新建文件夹 | `Ctrl + Shift + N` | `Cmd + Shift + N` | |
| 显示文件简介 / 属性 | `Alt + Enter` | `Cmd + I` | I = Info |
| 快速预览文件 | 无（需装 QuickLook 插件） | `Space`（空格） | Mac 杀手级功能——选中文件按空格就能预览内容，不用打开 |
| 显示隐藏文件 | 文件夹选项里勾选 | `Cmd + Shift + .` | 再按一次隐藏回来 |

## 2. 文本编辑

这一块是**肌肉记忆改得最痛苦的**，因为 Mac 的 Home/End/翻页行为和 Windows 完全不同。

### 2.1 光标移动

| 操作 | Windows | Mac |
|------|---------|-----|
| 行首 / 行尾 | `Home` / `End` | `Cmd + ←` / `Cmd + →` |
| 按词跳转 | `Ctrl + ←` / `Ctrl + →` | `Option + ←` / `Option + →` |
| 页首 / 页尾 | `Ctrl + Home` / `Ctrl + End` | `Cmd + ↑` / `Cmd + ↓` |
| 上一页 / 下一页 | `PageUp` / `PageDown` | `Fn + ↑` / `Fn + ↓`（或 `Option + PageUp/PageDown`，如果键盘有这些键） |

### 2.2 文本选择

> 规律：在移动快捷键基础上加 `Shift`，就是选中。

| 操作 | Windows | Mac |
|------|---------|-----|
| 选到行首 / 行尾 | `Shift + Home` / `Shift + End` | `Cmd + Shift + ←` / `Cmd + Shift + →` |
| 按词选中 | `Ctrl + Shift + ←/→` | `Option + Shift + ←/→` |
| 选到页首 / 页尾 | `Ctrl + Shift + Home/End` | `Cmd + Shift + ↑/↓` |
| 全选 | `Ctrl + A` | `Cmd + A` |

### 2.3 删除

| 操作 | Windows | Mac |
|------|---------|-----|
| 删除光标前一个字符 | `Backspace` | `Delete`（Mac 键盘上叫 delete，功能同 Backspace） |
| 删除光标后一个字符 | `Delete` | `Fn + Delete`（或外接键盘上的 `Del`） |
| 删除光标前一个词 | `Ctrl + Backspace` | `Option + Delete` |
| 删除光标后一个词 | `Ctrl + Delete` | `Fn + Option + Delete` |
| 删除到行首 | 无标准快捷键 | `Cmd + Delete`（大多数文本编辑器支持） |
| 删除整行 | 无标准快捷键 | `Cmd + Shift + Delete`（部分编辑器） |

> ⚠️ Mac 原生键盘只有一个 Delete 键，行为 = Windows 的 Backspace。"真正的 Delete"（删右边）要靠 `Fn + Delete`。外接 Windows 键盘时，`Backspace` 和 `Del` 两个键的行为是对的。

## 3. 窗口和桌面

| 操作 | Windows | Mac |
|------|---------|-----|
| 切换应用窗口 | `Alt + Tab` | `Cmd + Tab` |
| 同一应用内切换窗口 | 无（Win 11 有） | `Cmd + ~`（波浪号，Tab 上面那个键） |
| 关闭当前窗口 | `Alt + F4` | `Cmd + Q`（退出应用）/ `Cmd + W`（关闭当前窗口，应用还在） |
| 最小化 | `Win + ↓` 或 `Win + M` | `Cmd + M` |
| 隐藏当前应用 | 无 | `Cmd + H` |
| 强制退出 | `Ctrl + Alt + Delete` → 任务管理器 | `Cmd + Option + Esc` 或 `Option + 右键 Dock 图标` |
| 锁屏 | `Win + L` | `Cmd + Ctrl + Q` |
| 显示桌面 | `Win + D` | `Fn + F11` 或设置触发角（鼠标甩到屏幕角落） |
| Mission Control（所有窗口鸟瞰） | `Win + Tab` | `Ctrl + ↑` 或 `F3`（三指上推触控板也行） |
| 应用窗口平铺（左右半屏） | `Win + ←/→` | 把窗口拖到屏幕边缘，或用第三方工具（Rectangle / Magnet） |

### 3.1 特别说明：关闭 vs 退出

Windows 用户最容易搞混的点——

- **关闭窗口（`Cmd + W`）**：关掉当前文档/标签，**应用还在**（Dock 下的点还在）
- **退出应用（`Cmd + Q`）**：整个程序退出（Dock 下的点消失）

> Windows 习惯是"关掉最后一个窗口 = 退出程序"。Mac 不是——你关了所有 Finder 窗口，Finder 还在运行。想退出用 `Cmd + Q`。

## 4. 截图

Mac 的截图比 Windows 强太多，值得单独列。

| 操作 | Windows | Mac |
|------|---------|-----|
| 全屏截图 | `PrintScreen` | `Cmd + Shift + 3` |
| 区域截图 | `Win + Shift + S` | `Cmd + Shift + 4`（拖拽选区） |
| 窗口截图 | `Win + Shift + S` 后再选窗口 | `Cmd + Shift + 4` 然后按 `Space`，点窗口 |
| 截图工具栏（录屏等） | 无 | `Cmd + Shift + 5` |

> `Cmd + Shift + 4` 是最常用的——选区截图直接存桌面，不用像 Windows 那样还要粘贴。

## 5. 浏览器

| 操作 | Windows | Mac |
|------|---------|-----|
| 新标签 | `Ctrl + T` | `Cmd + T` |
| 关闭标签 | `Ctrl + W` | `Cmd + W` |
| 恢复关闭的标签 | `Ctrl + Shift + T` | `Cmd + Shift + T` |
| 地址栏聚焦 | `Ctrl + L` 或 `Alt + D` | `Cmd + L` |
| 刷新 | `F5` | `Cmd + R` |
| 强制刷新 | `Ctrl + F5` | `Cmd + Shift + R` |
| 开发者工具 | `F12` | `Cmd + Option + I` |
| 页面内搜索 | `Ctrl + F` | `Cmd + F` |

## 6. 特殊按键对照

Mac 键盘上少了一些键，多了一些键：

| Windows 键 | Mac 替代 |
|-------------|---------|
| `PrintScreen` | `Cmd + Shift + 3/4/5`（截图） |
| `Insert` | 无。大多数 Mac 应用没有"插入/覆盖"模式 |
| `Home` / `End` | `Cmd + ←/→`（行首/行尾）；`Cmd + ↑/↓`（文档首/尾） |
| `PageUp` / `PageDown` | `Fn + ↑/↓` |
| `Backspace`（←） | Mac 键盘上叫 `Delete`，功能一样 |
| `Delete`（删右边） | `Fn + Delete` |
| `Alt` | `Option (⌥)` |
| `Win` | `Cmd (⌘)` |
| `Ctrl` | `Ctrl (⌃)`——Mac 也有 Ctrl 键，但**大部分快捷键里它被 Cmd 替代了**。终端里 Ctrl 仍然活跃（`Ctrl+C` 终止、`Ctrl+A` 行首） |
| `AltGr`（欧洲键盘） | 无 |

## 7. 触控板手势（Windows 用户最容易忽略的 Mac 优势）

如果你是 MacBook，触控板的手势比快捷键还快：

| 手势 | 功能 |
|------|------|
| 三指上推 | Mission Control（所有窗口鸟瞰） |
| 三指左右划 | 切换桌面 / 全屏应用 |
| 四指捏合 | 启动台（Launchpad，所有应用图标） |
| 四指张开 | 显示桌面 |
| 双指轻点 | 右键 |
| 双指滚动 | 和 Windows 方向相反（Mac 默认"自然滚动"，类似手机触屏） |

> 如果觉得滚动方向反直觉：系统设置 → 触控板 → 滚动与缩放 → 关掉「自然滚动」。这就是 Mac 版的"鼠标滚轮反向"。

## 8. 必改的系统设置（让 Mac 更接近你的习惯）

刚上手建议立刻调的几项：

1. **Dock 自动隐藏**：系统设置 → 桌面与程序坞 → 自动隐藏和显示程序坞。给屏幕腾空间。
2. **触发角**：系统设置 → 桌面与程序坞 → 触发角（底部）。鼠标甩到屏幕角落触发：锁屏、显示桌面、Mission Control 等——替代 Windows 右下角显示桌面的习惯。
3. **三指拖移**：系统设置 → 辅助功能 → 指针控制 → 触控板选项 → 启用拖移 → 三指拖移。移动窗口不用按住再拖。
4. **键盘重复速率**：系统设置 → 键盘 → 按键重复速率拉到最快、重复前延迟拉到最短。Mac 默认按键连续输入很慢，改完之后按住一个键的反应才和 Windows 一样。

## 9. 怎么练肌肉记忆

不要试图一天全记住。策略：

1. **第一周只练一个转换**：把 `Ctrl` 换成 `Cmd`。每当你下意识按 Ctrl 时，纠正自己按 Cmd。这是最大的迁移。
2. **第二周练文本操作**：写代码时强迫自己用 `Cmd + ←/→` 跳行首行尾，不用触控板拖。
3. **截图、Finder 操作**：遇到时查表即可，慢慢就记住了。
4. **把这份文档随手放在旁边**。用的频率自然会降低。

> 两周后你会发现 Cmd 在大拇指位置比 Ctrl 在小指位置舒服太多了，回不去了。
