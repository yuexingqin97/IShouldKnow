# 终端美化方案

> 目标：让终端**好看、好用、信息清晰**。好看让你更愿意用，好用让你效率更高。

---

## 0. 现状摸底

先弄清楚你目前有什么，打开 iTerm2 依次敲：

```bash
echo $SHELL          # 看看你的 shell 是什么（大概率 /bin/zsh）
zsh --version        # zsh 版本
sw_vers              # macOS 版本
```

记下结果，后面可能会用到。

## 2. 技术选型

### 2.1 Shell 框架：Oh My Zsh

- **是什么**：zsh 的配置框架，管理插件、主题、配置
- **为什么选它**：社区最大、中文教程最多、插件生态最丰富
- **替代品**：starship（更轻量，跨 shell）、prezto（更轻更快但插件少）

> 建议从 Oh My Zsh 开始，入门最简单。以后想折腾再换。

### 2.2 主题：Powerlevel10k

- **是什么**：一个 zsh 主题，显示 git 分支、命令状态、当前目录等信息
- **为什么选它**：开箱即用的交互式配置向导（敲 `p10k configure`），不需要手写配置文件；显示 git 状态很实用
- **替代品**：spaceship（更简洁）、agnoster（经典但需要额外字体）、pure（极简）

### 2.3 字体：MesloLGS NF（Nerd Font）

- **是什么**：带有"图标字符"的等宽字体
- **为什么需要**：Powerlevel10k 的主题图标依赖这些特殊字符，普通字体显示不了
- Powerlevel10k 官方推荐 MesloLGS NF，安装最简单

### 2.4 配色方案

iTerm2 自带一批配色预设，新手建议：
- **Solarized Dark**：最经典，护眼，大部分工具的配色都适配了它
- **Dracula**：深色系，对比度高，流行
- **Snazzy**：鲜艳，年轻感

> 配色是非常主观的。到时候切几个看看自己喜欢哪个就行，不影响功能。

### 2.5 必备插件（Oh My Zsh 内置）

| 插件 | 干嘛的 | 为什么装 |
|------|--------|---------|
| `git` | git 别名 + 分支名显示 | 默认就开了，别关 |
| `zsh-autosuggestions` | 灰色提示历史命令 | 敲过一遍的命令不用再敲第二遍 |
| `zsh-syntax-highlighting` | 命令敲对变绿、敲错变红 | 实时语法检查，防呆 |
| `z` | 智能跳转目录 | `z project` 直接跳到 ~/Projects/xxx |

后两个需要额外安装，但 Oh My Zsh 插件管理会自动处理。

## 3. 实施步骤（按顺序来）

### 第一步：装 Oh My Zsh

```bash
# 官方安装命令，会下载脚本并执行
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

装完之后，你的终端外观会立刻变化（默认主题生效）。配置文件在 `~/.zshrc`。

### 第二步：装字体

1. 打开 [Nerd Fonts 下载页](https://www.nerdfonts.com/font-downloads)
2. 搜索 `Meslo`，下载 MesloLGS NF 的四个变体（Regular、Bold、Italic、Bold Italic）
3. 解压后双击每个 `.ttf` 文件，Mac 会弹出字体预览窗口，点「安装字体」
4. 打开 iTerm2 → Preferences (`Cmd + ,`) → Profiles → Text → Font → 选 **MesloLGS NF**

### 第三步：切换 iTerm2 配色

1. iTerm2 → Preferences → Profiles → Colors → Color Presets
2. 试试 Solarized Dark / Dracula / Snazzy
3. 选个看着舒服的

> 如果没有这些预设，去 [iTerm2-Color-Schemes](https://iterm2colorschemes.com/) 下载，然后 Import 到 Color Presets 里。

### 第四步：装 Powerlevel10k 主题

```bash
# 克隆到 Oh My Zsh 的自定义主题目录
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git \
  ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
```

然后编辑 `~/.zshrc`，找到 `ZSH_THEME=` 这一行，改成：

```bash
ZSH_THEME="powerlevel10k/powerlevel10k"
```

保存后重启 iTerm2，Powerlevel10k 的配置向导会自动弹出来。**按向导一步步选就行**——它会问你：
- 要不要图标？→ Yes
- 提示符风格？→ 选你看着顺眼的
- git 状态要不要显示？→ Yes
- 时间要不要？→ 看你自己

### 第五步：装插件

```bash
# zsh-autosuggestions（历史命令灰色提示）
git clone https://github.com/zsh-users/zsh-autosuggestions \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

# zsh-syntax-highlighting（命令语法高亮）
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

然后编辑 `~/.zshrc`，找到 `plugins=(git)` 这一行，改成：

```bash
plugins=(git zsh-autosuggestions zsh-syntax-highlighting z)
```

保存后执行 `source ~/.zshrc` 让配置生效。

## 4. 最终效果预期

完成后你的终端大概是这样的：

```
╭─ yxqin@MacBook  ~/Projects/IShouldKnow   main ≡ ──╮
│                                                          │
│  $ git status                      ← 你敲的命令（绿色）  │
│  On branch main                    ← 输出                │
│  nothing to commit                                     │
│                                                          │
│  # 敲错的命令会变红，正确的变绿                           │
│  # 敲 git 命令时灰色提示会显示你上次用的 git 子命令      │
│  # 提示符左边显示：用户、目录、git 分支、仓库状态         │
╰─────────────────────────────────────────────────────────╯
```

核心提升：
- **自动补全提示**：省掉 `↑` 翻历史的次数
- **语法高亮**：命令敲错立刻知道
- **git 信息一目了然**：在哪个分支、有没有未提交的改动
- **目录跳转更快**：`z ishouldknow` → 直接到家目录的项目里

## 5. 常见坑（先给你排雷）

> 这些都是我踩过的坑，你没必要再踩一遍。

| 坑 | 现象 | 解决 |
|----|------|------|
| 图标显示成方块/问号 | 装了 p10k 但图标是乱码 | 字体没切到 Nerd Font，去 iTerm2 设置里把字体改成 MesloLGS NF |
| `z` 插件不生效 | 敲 `z xxx` 说 command not found | 只在 Oh My Zsh 的 `plugins=(...)` 里加了还不够，需要 `source ~/.zshrc` 或重启终端 |
| 颜色和预期不一样 | p10k 的背景色和 iTerm2 配色打架 | 先设 iTerm2 配色，再跑 `p10k configure`，让 p10k 适配你的底色 |
| 终端变慢了 | 每次敲命令都卡一下 | 大概率是插件装太多或某个插件有性能问题，先只保留 git + autosuggestions + syntax-highlighting |

## 6. 要不要现在就搞？

建议学习顺序：

1. **先把 [[基础]] 里的命令敲熟**（1-2 天）——先会用，再美化
2. **装 Oh My Zsh + 换个配色**（5 分钟）——最小改动，立刻好看
3. **之后有空时装 p10k + 插件**（20 分钟）——功能性提升
4. **美化搞定后**：看 [[Neovim]]，开始终端写代码之路

> 不要一次全装上。一步步来，每步都确认没坏再继续。终端是你的工作环境，弄坏了你也得修——不如稳一点。

## 7. Windows 上能否复现？

能复现，但「终端程序」和「Shell 内部」是两个层，要分开看——正好帮你加深对 [[基础]] 里学到的概念的理解。

### 能原封不动复现的（和平台无关）

| 组件 | 跨平台？ | 说明 |
|------|---------|------|
| zsh + Oh My Zsh | ✅（走 WSL2） | zsh 在哪都一样，`.zshrc` 直接拷 |
| Powerlevel10k | ✅ | `.p10k.zsh` 直接拷，配置向导不用重跑 |
| 四个插件 | ✅ | 纯 zsh 插件，不沾操作系统 |
| MesloLGS NF 字体 | ✅ | 一个 `.ttf`，Windows Terminal 选上就行 |
| Dracula 配色 | ✅ | Windows Terminal 也有 Dracula 预设 |
| `.zshrc` / `.p10k.zsh` | ✅ | **配置文件直接拷过去，一模一样** |

### 需要换壳的（macOS 独占）

| 组件 | 能搬吗？ | Windows 替代 |
|------|---------|-------------|
| **iTerm2** | ❌ macOS 独占 | **Windows Terminal**（微软官方，现代好用，分屏/标签/配色/Nerd Font 全支持）或 **WezTerm**（Rust 跨平台，Lua 配置，一份文件两边通用） |

### 两条路

| 路线 | 复现度 | 怎么做 |
|------|-------|--------|
| **WSL2 + 直接拷配置** | **100%**，连 `.zshrc` 都不用改 | WSL2 里装 zsh → 拷配置 → Windows Terminal 里连进去。体验和 Mac 终端 95% 一致 |
| **原生 PowerShell + Oh My Posh** | 视觉 90%，底层不同 | 不用 zsh，用 PowerShell 7 + Oh My Posh（手动版 p10k）+ PSReadLine（补全/高亮）。能复现 Dracula + Nerd Font 图标 + git 信息，但插件和配置是独立的两套 |

> ⚠️ Oh My Posh ≠ Oh My Zsh。Oh My Posh 是**纯提示符渲染引擎**（只管那一行长什么样），不管插件。你 Mac 上的 `zsh-autosuggestions`、`zsh-syntax-highlighting` 这些是 zsh 自己的能力 + Oh My Zsh 管的，OMP 不碰。

### 如果你想「一次配置，到处用」

把 Mac 的 iTerm2 换成 **WezTerm**：它用 Lua 配置，一份 `wezterm.lua` 在 Mac 和 Windows 上通用，配合 WSL2 里的 zsh，从终端到 Shell 到配色全部可复现。你现在刚配好 iTerm2，不用换——等真有 Windows 机器那天再搞就行。

> 核心认知：**Shell 体验（zsh + p10k + 插件）靠 WSL2 原封不动复现，变的只是终端外壳。** 这和「VSCode 的智能来自 LSP 不是编辑器本身」是同一个道理——界面是壳，能力是核。
